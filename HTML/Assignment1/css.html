CSS (Cascading Style Sheets) controls the design and layout of web pages.
It is used to style HTML elements like colors, fonts, spacing, and positioning.
CSS makes websites attractive and user-friendly.
It supports selectors, classes, and IDs to target elements.
With CSS3, features like animations, transitions, and flexbox were introduced.
It separates content (HTML) from presentation (CSS).
Modern web design relies heavily on responsive CSS using media queries.
